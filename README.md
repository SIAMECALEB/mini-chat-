# Mini Chat Application

A simple mini chat application built to practice JavaScript fundamentals, DOM manipulation, and basic front-end logic as part of **The Odin Project** curriculum.

## 🔗 Live Demo 
📂 https://github.com/SIAMECALEB/mini-chat-

## 📌 About the Project
Mini Chat is a lightweight chat-style application that allows users to send and display messages on the page.  
The project focuses on handling user input, updating the DOM dynamically, and structuring JavaScript logic cleanly.

This project was built to strengthen:
- JavaScript fundamentals
- DOM manipulation
- Event handling
- Front-end project structure

## 🛠️ Built With
- HTML5  
- CSS3  
- JavaScript  
- Git & GitHub  

## ✨ Features
- Send messages through an input field
- Messages appear instantly on the screen
- Simple and clean user interface
- Beginner-friendly JavaScript logic

## 📚 What I Learned

- Manipulating the DOM dynamically
- Structuring JavaScript code
- Improving front-end problem-solving skills
- Using GitHub to manage a project repository
